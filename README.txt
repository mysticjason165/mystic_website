React + Tailwind Starter (minimal)
- Uses Vite + React (not included). To run locally:
  1. npm install
  2. npm install -D tailwindcss postcss autoprefixer
  3. npx tailwindcss init -p
  4. Configure tailwind and add @tailwind directives to src/styles.css
  5. npm run dev
